# Unnamed Platformer
## A platformer

<br/>

### Compilation Instructions:
-----------------------------------------------
## Dependencies
1. A C++ compiler
2. Raylib <br/> 

## To install all dependencies in one command:
Arch Based Linux Distros:
  ```
pacman -S raylib gcc
  ```

## To actually compile it
  ```
g++ main.cpp -lraylib -O3
  ```

<br/>
If you want to compile this on Windows, Have fun.
